# YoutubAPI 

## Feature

This API is return youtube playlist's music title and video ids.
Deploy to DETA.


## Requirement

* Python 3
* fastapi
* google-api-python-client

## Usage

Please post to https://3x0oij.deta.dev/ with youtube playlistId.

And please fill in APIkey.
